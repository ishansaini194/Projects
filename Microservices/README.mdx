# Microservices Architecture with gRPC and GraphQL

This project demonstrates a **microservices architecture** using:

* **gRPC** for inter-service communication
* **GraphQL** as the API gateway

It includes services for:

* Account management
* Product catalog
* Order processing

---

# 🏗 Project Structure

The project consists of the following main components:

* **Account Service**
* **Catalog Service**
* **Order Service**
* **GraphQL API Gateway**

---

| Service | Database      |
| ------- | ------------- |
| Account | PostgreSQL    |

# 📦 gRPC File Generation (Latest Setup)

---

## 1️⃣ Install Latest protoc (Recommended Method)

```bash
PROTOC_VERSION=$(curl -s https://api.github.com/repos/protocolbuffers/protobuf/releases/latest | grep tag_name | cut -d '"' -f 4 | sed 's/v//')

wget https://github.com/protocolbuffers/protobuf/releases/download/v${PROTOC_VERSION}/protoc-${PROTOC_VERSION}-linux-x86_64.zip

unzip protoc-${PROTOC_VERSION}-linux-x86_64.zip -d protoc

sudo mv protoc/bin/protoc /usr/local/bin/

protoc --version
```

---

## 2️⃣ Install Latest Go gRPC Plugins (Version Pinned)

> Always pin versions in real production projects.

```bash
go install google.golang.org/protobuf/cmd/protoc-gen-go@v1.33.0
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@v1.3.0
```

Verify:

```bash
protoc-gen-go --version
protoc-gen-go-grpc --version
```

---

## 3️⃣ Ensure Go Bin is in PATH

```bash
echo $PATH
export PATH="$PATH:$(go env GOPATH)/bin"
source ~/.bashrc
```

If using zsh:

```bash
source ~/.zshrc
```

---

## 4️⃣ Create `pb` Folder

```bash
mkdir -p pb
```

---

## 5️⃣ Update `account.proto` (Modern Style)

Instead of:

```proto
option go_package = "./pb";
```

Use:

```proto
option go_package = "your-module-name/pb;pb";
```

Example:

```proto
option go_package = "github.com/yourusername/project/account/pb;pb";
```

⚠️ This avoids import issues in modern Go modules.

---

## 6️⃣ Generate gRPC Files (Modern Recommended Flags)

```bash
protoc \
  --go_out=pb \
  --go_opt=paths=source_relative \
  --go-grpc_out=pb \
  --go-grpc_opt=paths=source_relative \
  account.proto
```

If multiple proto files:

```bash
protoc \
  --go_out=pb \
  --go_opt=paths=source_relative \
  --go-grpc_out=pb \
  --go-grpc_opt=paths=source_relative \
  *.proto
```

# 🔮 GraphQL API Usage

The GraphQL API provides a unified interface to interact with all microservices.

---

## 🔎 Query Accounts

```graphql
query {
  accounts {
    id
    name
  }
}
```

---

## ➕ Create an Account

```graphql
mutation {
  createAccount(account: {name: "New Account"}) {
    id
    name
  }
}
```

---

## 📦 Query Products

```graphql
query {
  products {
    id
    name
    price
  }
}
```

---

## ➕ Create a Product

```graphql
mutation {
  createProduct(product: {
    name: "New Product",
    description: "A new product",
    price: 19.99
  }) {
    id
    name
    price
  }
}
```

---

## 🛒 Create an Order

```graphql
mutation {
  createOrder(order: {
    accountId: "account_id",
    products: [
      {id: "product_id", quantity: 2}
    ]
  }) {
    id
    totalPrice
    products {
      name
      quantity
    }
  }
}
```

---

## 👤 Query Account with Orders

```graphql
query {
  accounts(id: "account_id") {
    name
    orders {
      id
      createdAt
      totalPrice
      products {
        name
        quantity
        price
      }
    }
  }
}
```
